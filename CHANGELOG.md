# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-28

### Added
- Initial release
- Horizontal scroll functionality using scroll-jacking technique
- Configurable section count, width, and scroll speed
- Mobile responsive with optional disable
- Lightweight vanilla JavaScript implementation
- No external dependencies
- Multiple width variants (50%, 60%, 70%, 80%, 90%, 100%)
- Smooth CSS transform animations
- Automatic height calculation
- Resize event handling

### Features
- Vertical scroll controls horizontal section movement
- Sticky positioning during scroll
- Customizable scroll speed multiplier
- Mobile stacking option
- Multiple horizontal scroll groups on same page
- Performance optimized with throttled scroll events
